Stack Overflow Clone

A full-stack Q&A platform built to replicate the core functionality of Stack Overflow. This project focuses on complex data relationships, real-time interactions, and robust state management.


Features : 

Authentication: Secure login and signup powered by Appwrite Auth.

Ask Questions: Users can post questions with detailed descriptions using a rich text editor.

Voting System: Upvote or downvote questions and answers to highlight the best content.

Top Contributors: A leaderboard system to track the most active users.

Search & Tags: Efficiently find questions using specific tags or keywords.

Global State: Seamless UI updates and data flow managed by Zustand.

Responsive Design: Fully mobile-responsive interface built with Tailwind CSS.



🛠️ Tech Stack

Frontend: Next.js (React Framework)

Backend-as-a-Service: Appwrite (Database, Auth, and Storage)

State Management: Zustand

Styling: Tailwind CSS & Shadcn/UI

Icons: Lucide React
